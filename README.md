# StudyMaster
AI Study Master is a web app that helps students organize their semester by analyzing uploaded course documents like syllabi, schedules, and assignment lists. It extracts key information such as deadlines, exams, and topics, then generates a structured plan with tasks, timelines, and AI-driven study recommendations for each assignment and exam.
